# Project Brief — Garments BOM OCR Management System

## What This System Does

Garments BOM OCR Management System is a centralized business workflow platform for garment manufacturing operations.

It helps teams manage the full order-to-shipment process from one controlled workspace instead of depending on many separate Excel files.

## Business Problem

Before this software, a company may use 40-50 Excel files for BOM, purchase, store, commercial, accounts, shipment, and approval follow-up.

Common problems:

- Same data entered multiple times
- File version confusion
- Slow department communication
- Manual mistakes
- Missing responsibility tracking
- Management cannot see real-time progress clearly
- PO, store, commercial, and shipment follow-up becomes difficult

## Software Benefit

This system helps by:

- Keeping order and BOM data in one place
- Connecting department-wise workflow
- Reducing duplicate entry
- Improving visibility for management
- Making approval and follow-up easier
- Supporting PDF/Excel/report output
- Making each user work only on responsible areas
- Creating a clear order-to-shipment process

## Main Users

- Merchandising team
- Commercial team
- Store team
- Accounts team
- Management/Admin team

## Main Business Areas

- Buyer and style information
- Order entry
- BOM preparation
- OCR-assisted document/data entry
- Review and approval
- Purchase and PO process
- Store receiving and inventory update
- Commercial document coordination
- Accounts/payment follow-up
- Shipment tracking
- Dashboard and reporting
